# Java Quiz Application

## Overview
This project is a console-based Quiz Application developed using Java. It allows users to attempt multiple-choice questions and receive a score based on their performance. The application demonstrates core Java concepts such as file handling, loops, conditionals, and user input handling.

---

## Features
- Multiple-choice quiz system  
- Real-time score calculation  
- File handling for storing and retrieving scores  
- Option to attempt multiple quizzes  
- Input validation for better user experience  


## Technologies Used
- Java  
- File I/O (BufferedReader, FileWriter)  
- OOP Concepts
